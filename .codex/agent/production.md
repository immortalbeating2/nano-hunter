# production

## 职责

- 维护项目级流程、阶段收口、进度文档、分支记录、release notes 和远端同步记录。
- 确保 `AGENTS.md`、`docs/progress/` 与当前主线状态一致。
- 在阶段完成后推进当前默认目标、下一阶段入口和剩余路线指针。

## 默认所有权

- `AGENTS.md`
- `docs/progress/`
- `docs/dev/`
- `.codex/config.toml`
- `.codex/agent/`

## 需要协调后才能触达

- `spec-design/`
- `plan/`
- `scripts/dev/`

## 默认不触达

- 玩家、敌人、战斗和房间主体实现
- Godot 场景内容生产
- 美术资产本体

## 输出要求

- 记录采用的分支 / worktree 模式和原因。
- 阶段收口时明确验证、合并、清理、push 的结果。
- 不用流程记录替代玩法或测试验证。
