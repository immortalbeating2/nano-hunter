# Nano Hunter — 设计规格书 (2026-03-23)

## 1. 概述
一款类银河恶魔城游戏，参考《空洞骑士》的探索与战斗结构，结合《Ori》的视觉柔和感与《山海经》的生物设计。时代背景为南北朝时期。主角 **Luna** 是镇妖卫中的一名赏金猎人，隐藏着佛妖混血的身份。其独特能力：因妖性血统可吸收并融合元素之力，通过佛门符印加以引导。游戏核心玩法：高度自由的元素序列连锁与符印姿态切换。

---

## 2. 核心战斗与元素系统

### 元素池（6–8 个）
山海经意象元素（示例）：风、雷、火、水、山、木、金、影/魇。

### 符印姿态（4–5 个）
姿态代表"施力方式/心法"，决定技能形态与节奏（如：疾、御、咒、化、断）。姿态切换是核心机制。

### 序列连锁机制
- 玩家切换姿态 → 释放技能 → 积累"元素序列"。
- 同元素不同顺序产生不同效果（如：风→雷 与 雷→风 形态不同）。
- 序列窗口：连段中持续记录 2–3 次元素事件，超时则清空。

### 反应层级
- **基础反应**：元素对元素的核心效果（击退、易伤、破甲、定身等）。
- **序列修饰**：第二/第三步改变范围、方向、持续时间或倍率。

### 节奏与轮转
- 战斗以"轮转"为主，姿态切换有短冷却或资源消耗，防止无脑刷序列。

### 分章节解锁
- 早期：2–3 个元素 + 2 个姿态。
- 中期：完整元素池逐步解锁，元素变种激发成为核心。
- 后期：妖性觉醒解锁深层连锁。

---

## 3. 主角：Luna
- **公开身份**：镇妖卫赏金猎人。
- **隐藏身份**：佛门与妖的混血后裔。
- **符印来源**：佛门秘籍中习得的符印之法。
- **独特能力**：因体内妖性，可吸收并融合元素力量，驱动符印产生连锁反应。
- **叙事钩子**：
  - 执行悬赏任务，逐步揭露官僚腐败与军阀叛乱。
  - 面对人类与妖族双方对"妖"的偏见。
  - 妖性觉醒提供更强力的替代战斗路线。

**视觉锚点**：面具/法器（赏金猎人标识与技能媒介）、符印法阵、灵力链序列可视化。妖性觉醒时带有独特视觉异变。

---

## 4. 世界观与势力

### 背景
南北朝战乱频仍，妖怪滋生。佛门因庇护百姓而深受爱戴，皇室忌惮其影响力。

### 势力关系

| 势力 | 定位 | 与 Luna 的关系 |
|------|------|---------------|
| **佛门正道** | 独立宗教，战乱中庇护百姓，深得民心 | 符印来源，不知 Luna 的妖性血统与赏金身份 |
| **妖怪势力** | 大妖统领的山海经异兽集群 | 猎杀目标，部分大妖与 Luna 妖性有共鸣 |
| **镇妖卫（赏金猎人）** | 皇家贴榜招揽的赏金猎人组织 | Luna 的公开身份 |
| **腐化官僚集群** | 与妖怪交易、压榨百姓血肉、贪图享乐 | 压迫者，民间苦难的直接制造者 |
| **地方军阀** | 战乱发起者，觊觎推翻皇权，常与妖怪/官僚暗中勾结 | 乱源 |
| **皇家** | 皇权中心，与佛门微妙博弈 | 借镇妖卫制衡佛门与维持秩序 |

## 5. 成长与进阶系统

- **元素槽位**：击败特定妖怪并吸收其元素之力，逐步解锁槽位。
- **元素变种激发**：特定元素组合可触发变种效果（如：雷+火 → 爆焰，范围扩大并附加灼烧），作为探索与熟练度的奖励。
- **符印进阶**：佛门秘籍分层级，完成章节任务或击败 Boss 解锁新符印/姿态。
- **妖性觉醒**：满足特定条件（收集/抉择/隐藏任务）后解锁隐藏姿态与深层元素组合。
- **技能树**：
  - 每个元素一张子图，每个姿态一张子图，序列组合作为连接层。
  - 部分强力组合需要同时点亮元素 + 姿态的特定节点。
- **佛门圣物**：地图探索奖励，如念珠、法轮、铜铃等，可镶嵌至装备，提供被动强化与元素亲和加成。
- **大妖组件**：击败大妖后，其身体组件散落于地图（如：烛龙鳞片、九尾狐尾、穷奇爪等），作为稀有探索奖励，用于制作或强化特殊装备。
- **装备/法器**：
  - 面具/法器随流程升级，强化基础数值（攻击、生命、元素亲和等）。
  - 装备孔位可镶嵌圣物/大妖组件，进一步定制 Build。
- **珍品交易市集**：收集的珍奇物品（多余圣物、妖怪材料、探索碎片等）可在市集交易，换取稀有配方、限时物品或情报。部分特殊物品只能通过交易获取。
- **数值成长**：克制膨胀（空洞骑士式），强调技术而非刷数值。

## 6. 关卡与探索结构

- **地图架构**：类银河恶魔城非线性大地图，多个可自由往返的区域，通过能力解锁实现回溯。
- **区域设计**（规划 6–8 个核心区域）：
  - 山门古刹（教程区）
  - 雷泽荒原
  - 赤焰熔岭
  - 幽冥黄泉
  - 洪荒戈壁
  - 苍穹龙渊
  - 皇城暗面
  - 深渊妖域
- **能力门控**：特定元素能力解锁新通路（如：火融化冰障、风吹散毒雾），驱动探索循环。
- **存档点**：官府驿站（镇妖卫标准休息点）。
- **悬赏榜**：每个驿站配有悬赏榜，提供赏金任务、情报交换、物品兑换。
- **佛门石碑**：叙事碎片载体，散布于地图，呈现南北朝佛妖冲突背景（非存档机制）。
- **支线与隐藏**：每个区域含 1–2 条支线与隐藏区域，提供圣物/组件/妖性觉醒条件。

## 7. 叙事结构

### 主线
Luna 以镇妖卫身份执行悬赏任务，逐步发现：
- 腐化官僚与妖怪交易血肉百姓的黑幕。
- 地方军阀与妖怪结合滋生叛乱。
- 皇家血脉流落民间，一名皇子从小携带"灭妖神器"。

最终揭开战乱幕后黑手，直面最终 Boss。

### 主题
身份认同、偏见与羁绊、人性与妖性的力量平衡。

### 结局方向

| 结局 | 走向 | 代价 |
|------|------|------|
| **妖性完全觉醒** | 拥抱妖性，获得完整力量 | 失去佛门容身之所 |
| **佛妖平衡** | 找到共处之道 | 需极高代价换取 |
| **皈依佛门不问俗世** | 斩断妖性与尘缘，投入佛门修行 | 放弃一切非常之力 |
| **归隐山田** | 放弃力量，与落魄皇子一同归隐 | 退出所有纷争 |

---

## 8. 核心玩法循环

1. **驿站接榜** → 选择悬赏目标/区域
2. **前往区域探索** → 解锁通路/发现线索
3. **遭遇妖怪** → 战斗并吸收元素/收集材料
4. **返回驿站** → 兑换/制作/升级/解锁新任务
5. **推进主线** → 解锁新区域/新姿态/新元素

## 9. 视觉与美术方向

- **整体基调**：Ori 的柔美流动感 × 山海经的东方神秘感。
- **色彩方案**：
  - 主调：墨青、黛紫、月白、岩赤（东方水墨/工笔色系）。
  - 元素色：每个元素有对应色系（雷=紫蓝、火=橙红、风=青绿等）。
  - 妖性觉醒时切换为更浓烈的对比色调。
- **渲染风格**：
  - 2D 手绘风格（Godot 2D + Shader 实现水墨/光晕效果）。
  - 背景层叠营造纵深感（类 Ori 的视差）。
  - 粒子系统承载元素效果与灵力链可视化。
- **动画风格**：流畅、有韵律感，强调 Luna 的灵动感与法器特效。
- **UI 设计**：
  - 符印姿态以法阵 HUD 呈现。
  - 元素序列以灵力链视觉追踪。
  - 驿站/悬赏榜采用竹简/卷轴风格 UI。
- **光照**：柔和体积光 + 元素粒子辉光，营造奇幻氛围。

## 10. 开发优先级与下一步

### 核心范围（优先构建）
1. 战斗循环基础（移动、攻击、受击）
2. 符印姿态切换系统
3. 元素序列连锁机制
4. 基础 UI（HUD、驿站、悬赏榜）
5. 一个可玩的区域原型（教程区）

### 延伸范围（后续迭代）
- 完整元素池与变种激发
- 成长系统（技能树、圣物、组件）
- 珍品交易市集
- 叙事与多结局
- 全区域内容填充
